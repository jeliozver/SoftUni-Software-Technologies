Calculator - Apache Maven + Spring Web App
========================

Features
------------

* Arithmetic Operators;
	* Addition
	* Subtraction
	* Division
	* Multiplication
	* Modulo operation
	* Exponentiation
 
* Bitwise Operators;
	* And
	* Or
	* Xor
	* Not
	* Shift left
	* Shift right

* Misk Operators;
	* X percent of Y
	* X is what percent of Y
	* Square root of X
	* X factorial
 
* Basic error handling;
